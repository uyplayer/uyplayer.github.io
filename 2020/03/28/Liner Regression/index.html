Liner Regression

uyplayer
Machine Learning

发布于：2020年3月28日
次浏览

字数：2.3k字

我们生活在海量数据，功能强大的计算机和人工智能的时代。 这仅仅是个开始。 数据科学和机器学习正在推动图像识别，自动驾驶汽车的发展，金融和能源领域的决策，医学的进步，社交网络的兴起等等。 线性回归是其中的重要部分。
线性回归是基本的统计和机器学习技术之一。 无论你要进行统计，机器学习还是科学计算，都有很大的机会需要它。 建议先学习它，然后再尝试更复杂的方法。

什么是回归？

  • 回归搜索变量之间的关系。

例如,可以观察某家公司的几名员工，并尝试了解他们的薪水如何取决于功能，例如经验，学历，职位，所工作的城市等。
这是一个回归问题，其中与每个员工相关的数据代表一个观察值。假定经验，学历，角色和城市是独立的特征，而薪水则取决于它们。
同样，你可以尝试建立房屋价格对房屋面积，卧室数量，到市中心的距离等的数学依赖性。
通常，在回归分析中，你通常会考虑一些有趣的现象并有很多观察结果。每个观察都具有两个或多个特征。假定（至少）一个功能依赖于其他功能，然后尝试在它们之间建立关系。
换句话说，你需要找到一个将某些功能或变量很好地映射到其他功能的函数。
因果关系称为因变量，输出或响应。
独立特征称为独立变量，输入或预测变量。
回归问题通常具有一个连续且无界的因变量。但是，输入可以是连续的，离散的，甚至是分类数据，例如性别，国籍，品牌等。
通常的做法是用y表示输出，用x表示输入。如果有两个或多个自变量，则可以将它们表示为向量𝐱=（𝑥₁，…，𝑥ᵣ），其中𝑟是输入数。

你何时需要回归？

通常需要回归以回答某种现象是否以及如何影响另一种现象或如何关联多个变量。 例如，你可以使用它来确定经验或性别是否影响工资以及工资的影响程度。
当你要使用一组新的预测变量预测响应时，回归分析也很有用。 例如，你可以尝试根据室外温度，一天中的时间以及该家庭中的居民人数来预测下一个小时的家庭用电量。
回归用于许多不同的领域：经济，计算机科学，社会科学等等。 随着大量数据的可用性和对数据实用价值的认识的提高，其重要性每天都在提高。

什么是线性回归？

  • 线性：两个变量之间的关系是一次函数关系的——图象是直线，叫做线性。
  • 非线性：两个变量之间的关系不是一次函数关系的——图象不是直线，叫做非线性。
  • 回归：人们在测量事物的时候因为客观条件所限，求得的都是测量值，而不是事物真实的值，为了能够得到真实值，无限次的进行测量，最后通过这些测量数据计算回归到真实值，这就是回归的由来。

问题表述

  • 在一组独立变量𝐱=（𝑥₁，…，𝑥ᵣ）上实现某些因变量linear的线性回归时，其中𝑟是预测变量的数量，您假设𝑦和x之间存在线性关系：𝑦= 𝛽₀ + 𝛽₁𝑥₁ +⋯ + 𝛽ᵣ𝑥ᵣ + 𝜀。 这个方程是回归方程。 𝛽₀，𝛽₁，…，𝛽ᵣ是回归系数，𝜀是随机误差。
  • 线性回归计算回归系数的估计值，或仅计算预测的权重，分别用𝑏₀，𝑏₁，…，𝑏ᵣ表示。 他们定义了估计的回归函数𝑓（𝐱）=𝑏₀+𝑏₁𝑥₁+⋯+𝑏ᵣ𝑥ᵣ。 该函数应该很好地捕获输入和输出之间的依赖关系。
  • 对于每个观察值𝑖= 1，…，𝑛，估计或预测的响应𝑓（𝐱ᵢ）应尽可能接近相应的实际响应𝑦ᵢ。 所有观测值𝑖= 1，…，𝑛的差𝑦ᵢ-𝑓（𝐱ᵢ）称为误差。 回归是关于确定最佳预测权重，即与最小误差相对应的权重。
  • 为了获得最佳权重，通常对于所有观测值𝑖= 1，…，𝑛，应使误差平方和（SSR:minimize the sum of squared residuals）最小化：SSR =Σᵢ（𝑦ᵢ-𝑓（𝐱ᵢ））²。 这种方法称为普通最小二乘法。
