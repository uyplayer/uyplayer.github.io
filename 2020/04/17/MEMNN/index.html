概括

Memory Networks是一种学习模型并它可以读取和写入的。 在机器学习训练模型中它要学习如何有效使用内存组件(memory componet)。
Memory Networks 结构：

  • memory m：是对象的索引数组（比如向量或字符串数组）
  • 输入映射I：它将输入转换为内部要数据
  • G(generalization)：在给定新输入数据的情况下更新旧内存。它更新的不是所有的内存，通过一种机制寻找需要更新的内存
  • O输出，在给定新输入和当前内存状态的情况下，将在特征表示空间中产生新输出。意识是在这里我们进行计算并形成输出
  • R：O输出的结果转换为所需的格式，例如文本响应或动作。这里的作用类似decoder

Text/QA(example)

  • I:利用标预处理，通过从文本转换特征向量，它也可以将输入编码为内部特征表示
  • G:引入函数S，该函数将I生成的特征表示映射到单个内存插槽(memory)，并仅更新S（I（x））处的内存。就是mN = x, N = N + 1
  • O：按照给定输入X，0（output module）它寻找k个相应的memory，当k=1时：这里有一个scoring函数s，它从memory里进行检索最符合我们输入x的m_i

当k=2时：开始scoring函数s寻找第二个相应的m

返回[x,m_01,m_02] 作为R的输入；

  • R: 该模块要输出text作为最后输出r；下面简单定义另一个scoring函数𝑆_𝑅；

比如：x= “Where is the milk now?；先O模块首先为所有memory评分，然后在之间存到内存数据里面开始检索最合适的𝑚_01 = “Joe left the milk” ，然后我们继续检索第二个最合适的𝑚_02 = “Joe travelled to the office” ；最后我们R模块通过上面的函数输出r=”office“
Scoring函数：前面说的两个模块(𝑠_𝑂 , 𝑠_𝑟)中scoring函数形式一样的；
